### To-Do App - todo-script.js

#### TodoApp Class

##### Constructor
```javascript
class TodoApp {
    constructor() {
        this.todos = this.loadTodos();      // Load from localStorage
        this.currentFilter = 'all';
        this.init();
    }
}
```

##### Main Methods

1. **init()**
   - Purpose: Initialize the app
   - Sets up event listeners and renders initial UI

2. **addTodo()**
   - Purpose: Add a new task
   - Parameters: None (uses input field value)
   - Returns: Void
   - Creates todo object with:
     - id: Unique timestamp
     - text: Task description
     - completed: Boolean (false)
     - createdAt: Creation timestamp

3. **removeTodo(id)**
   - Purpose: Delete a task
   - Parameters: `id` (Number/timestamp)
   - Returns: Void
   - Removes todo from array and updates storage

4. **toggleTodo(id)**
   - Purpose: Mark task as completed/uncompleted
   - Parameters: `id` (Number/timestamp)
   - Returns: Void
   - Toggles `completed` property

5. **clearCompleted()**
   - Purpose: Remove all completed tasks
   - Parameters: None
   - Returns: Void
   - Shows confirmation dialog
   - Filters out completed todos

6. **getFilteredTodos()**
   - Purpose: Get todos based on current filter
   - Parameters: None
   - Returns: Array of todos (filtered)
   - Filters based on:
     - 'all': Returns all todos
     - 'active': Returns only incomplete todos
     - 'completed': Returns only completed todos

7. **render()**
   - Purpose: Update the UI
   - Parameters: None
   - Returns: Void
   - Creates DOM elements for tasks
   - Shows/hides empty state
   - Updates statistics

8. **updateStats()**
   - Purpose: Update task count displays
   - Parameters: None
   - Returns: Void
   - Updates totalCount and completedCount
   - Enables/disables clear button

9. **saveTodos()**
   - Purpose: Save todos to localStorage
   - Parameters: None
   - Returns: Void
   - Stores todos array as JSON string

10. **loadTodos()**
    - Purpose: Load todos from localStorage
    - Parameters: None
    - Returns: Array of todos or empty array
    - Retrieves and parses JSON from storage

11. **escapeHtml(text)**
    - Purpose: Sanitize user input
    - Parameters: `text` (String)
    - Returns: Safe HTML string
    - Prevents XSS attacks

#### Event Listeners Setup
```javascript
setupEventListeners() {

    // Enter key to add task
```
